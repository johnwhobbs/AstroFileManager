# Claude Code Instructions for AstroFileManager

## Repository Structure
This is a Python astrophotography file management application.

## Exploring Files
- Use View on directory paths to see contents
- Use View on file paths to read files
- Use GlobTool to find files (e.g., "**/*.py" for all Python files)

## Coding Standards
- Python 3.8+
- Follow PEP 8
- Add docstrings to all functions
- Include type hints where helpful
- Add comments for complex logic
- Include basic error handling

## File Organization
- Main application code in root or src/
- UI components in ui/ directory
- Utilities and helpers organized by function

## When Making Changes
1. Explore the codebase first
2. Understand existing patterns
3. Match the existing code style
4. Test your changes if possible
5. Commit with clear messages
